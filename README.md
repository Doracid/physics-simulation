# Physics Simulation — 电磁物理仿真平台

基于 **Pygame** 和 **Python** 的交互式电磁物理仿真平台，支持静电场/磁场仿真、直流/交流电路分析、带电粒子动力学，含自由模式和学习模式。

## 功能概览

- **静电场仿真** — 点电荷、电偶极子、导体球/球壳/无限大金属板（镜像法）、均匀电场区域
- **静磁场仿真** — 条形磁铁、马蹄形磁铁、通电螺线管（15 偶极子模型）、载流直导线（Biot-Savart）
- **直流电路求解** — 修正节点分析（MNA），支持电阻、电容、电感、电源、螺线管，自动拓扑聚类
- **交流稳态求解** — 复阻抗相量法（complex MNA），自动检测纯交流电路并分发
- **带电粒子动力学** — 洛伦兹力 F=q(E+v×B)，Boris 辛积分器，轨迹记录
- **法拉第电磁感应** — 实时磁通量追踪，闭合回路/螺线管感应电动势，楞次定律
- **镜像法** — 导体球、导体球壳、无限大金属板的自动镜像电荷计算
- **学习模式** — 17 个预置实验项目，覆盖静电学、电路、磁学，可前后导航
- **自由模式** — 沙盒环境，所有元件可用，自由搭建电路与电磁场

## 项目结构

| 文件/目录 | 行数 | 说明 |
|-----------|------|------|
| `main.py` | ~3700 | 主入口 — Pygame GUI、主循环、事件处理、撤销/重做、学习模式 UI、求解器调度 |
| `elements.py` | ~2030 | 所有交互元件类（电荷、磁铁、导线、电阻、电源、电容器等） |
| `physics.py` | ~1260 | 物理引擎 — E/B 场计算、场线追踪（RK2）、镜像法、Biot-Savart、法拉第感应 |
| `circuit.py` | ~268 | 直流/暂态电路求解器（MNA，伴随模型，拓扑聚类） |
| `ac_solver.py` | ~228 | 交流稳态相量求解器（复阻抗 MNA） |
| `study/` | — | 17 个学习项目文件（.txt 格式） |

## UI 主题

**霓虹玻璃拟态**（Neon Glass Morphism），紫主青辅：

- 深黑背景（RGB 10,11,18）+ 角落紫/青光晕
- 圆角玻璃卡片（半透明紫色渐变 + 顶部高光）
- 按钮立体渐变（上亮下暗 + 玻璃高光 + 暗边）
- 发光描边（Cyan 选中发光）
- 自适应中文字体回退链
- 标题阴影 + 首页 Cyan 发光装饰条

## 开始使用

### 环境要求
- Python 3.8+
- Pygame 2.x
- NumPy

### 安装与运行
```bash
pip install pygame numpy
python main.py
```

### 学习模式
在首页点击「学习模式」进入，用顶栏「◀ 前一个」「下一个 ▶」浏览项目。完成所有项目后可选「重新学习」或「退出学习」。

## 技术要点

- **物理精度**：RK2 场线追踪、Boris 辛积分、MNA 电路求解、复相量 AC 分析
- **镜像法**：导体球/球壳/无限大平板的自动镜像电荷，用于场计算和场线追踪
- **法拉第感应**：逐帧 Bz 通量积分，楞次定律方向判定
- **螺线管模型**：15 个虚拟磁偶极子沿轴分布，3D Biot-Savart 积分
- **撤销系统**：JSON 快照，最多 50 步，播放前自动快照可还原
- **自动求解分发**：自动检测纯 AC / DC / 混合电路，调用对应求解器
